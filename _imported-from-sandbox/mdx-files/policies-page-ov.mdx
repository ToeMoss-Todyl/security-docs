<div id="policies-page-ov"></div>

The **Policies** page lists all of the integration policies configured for ((elastic-defend)). 

<DocCallOut title="Requirements">

You must have the **((elastic-defend)) Policy Management** <DocLink id="enSecurityEndpointManagementReq">privilege</DocLink> to access this feature.

</DocCallOut>

Click on an integration policy's name to configure its settings. For more information on configuring an integration policy, refer to <DocLink id="enSecurityConfigureEndpointIntegrationPolicy">Configure an integration policy for ((elastic-defend))</DocLink>.
