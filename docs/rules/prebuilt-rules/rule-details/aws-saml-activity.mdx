<div id="aws-saml-activity"></div>

Identifies when SAML activity has occurred in AWS. An adversary could manipulate SAML to maintain access to the target.

**Rule type**: query

**Rule indices**: 

* filebeat-*
* logs-aws*

**Severity**: low

**Risk score**: 21

**Runs every**: 5m

**Searches indices from**: now-25m ([Date Math format](((ref))/common-options.html#date-math), see also <DocLink id="serverlessSecurityRulesUiCreate" section="set-the-rules-schedule">`Additional look-back time`</DocLink>)

**Maximum alerts per execution**: 100
