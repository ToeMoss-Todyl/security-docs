<div id="execution-via-tsclient-mountpoint"></div>

Identifies execution from the Remote Desktop Protocol (RDP) shared mountpoint tsclient on the target host. This may indicate a lateral movement attempt.

**Rule type**: eql

**Rule indices**:

* winlogbeat-*
* logs-windows.*
* endgame-*
