<div id="aws-ec2-vm-export-failure"></div>

Identifies an attempt to export an AWS EC2 instance. A virtual machine (VM) export may indicate an attempt to extract or exfiltrate information.

**Rule type**: query

**Rule indices**: 

* filebeat-*
* logs-aws*

**Risk score**: 21
