<div id="microsoft-365-exchange-dlp-policy-removed"></div>

Identifies when a Data Loss Prevention (DLP) policy is removed in Microsoft 365. An adversary may remove a DLP policy to evade existing DLP monitoring.

**Rule type**: query

**Rule indices**: 

* filebeat-*
* logs-o365*

**Severity**: medium

**Risk score**: 47

**Runs every**: 5m
