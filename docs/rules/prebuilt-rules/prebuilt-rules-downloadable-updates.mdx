<div id="prebuilt-rules-downloadable-updates"></div>

This section lists all updates to prebuilt detection rules, made available with the **Prebuilt Security Detection Rules** integration in Fleet.

To update your installed rules to the latest versions, follow the instructions in <DocLink id="serverlessSecurityPrebuiltRulesManagement" section="update-elastic-prebuilt-rules">Update Elastic prebuilt rules</DocLink>.

For previous rule updates, please navigate to the [last version](https://www.elastic.co/guide/en/security/8.8/prebuilt-rules-downloadable-updates.html).
