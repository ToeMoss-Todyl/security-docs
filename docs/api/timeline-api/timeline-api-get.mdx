Get Timelines or Timeline templates.

## Request URL

`GET <kibana host>:<port>/api/timelines`

### Example requests

Get Timelines:

```console
GET api/timelines?page_size=10&page_index=1&sort_field=updated&sort_order=desc&timeline_type=default

```
