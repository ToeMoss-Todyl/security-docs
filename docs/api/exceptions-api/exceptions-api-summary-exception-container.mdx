Retrieves an exception container summary.

## Request URL

`POST <kibana host>:<port>/api/exception_lists/summary`

### Example requests

Gets the summary from an exception container:

```console
GET api/exception_lists/summary
{
  "filter": "filter-clause",
  "id": "allowed-process-id",
  "list_id": "allowed-processes",
  "namespace_type": "single",
}
```
{/* KIBANA */}

## Response code

`200` 
    : Indicates a successful call.

## Response payload

A summary of the exception container:

```json
{
    windows: 1,
    linux: 3,
    macos: 2,
    total: 6,
}
```

<DocCallOut title="Note">
For a host isolation exception, the values for `windows`, `linux`, `macos`, and `total` are all the same, as each host isolation exception entry applies to all operating systems.
For example:
</DocCallOut>

```json
{
    windows: 5,
    linux: 5,
    macos: 5,
    total: 5,
}
```
