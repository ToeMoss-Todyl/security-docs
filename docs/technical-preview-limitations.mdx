The following limitations apply to ((elastic-sec)) technical preview projects:

- Maximum data ingest rate, total of all data sources &mdash; 100GB per day
- Maximum number of ((ml)) jobs &mdash; 50
- Maximum Search boost window &mdash; 14 days
- Maximum searchable data size &mdash; 10TB
- Maximum number of endpoints and cloud assets for ((fleet)) and ((agent)) management &mdash; 40,000
